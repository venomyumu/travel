## 行程总说明

- 核心覆盖：[[大阪城公园]]、[[黑门市场]]、[[心斋桥-道顿堀]]
- 动线逻辑：不绕路、少折返、适合后续前往奈良 / 京都
- 住宿推荐：大阪难波 / 日本桥（美食 + 交通最方便）

---

# Day 1 大阪经典全景日

1. 上午：[[大阪城公园]]（游玩 + 打卡）
2. 中午：返回市区，午餐简餐
3. 下午：[[黑门市场]]（逛吃主力）
4. 傍晚+晚上：[[心斋桥-道顿堀]]（闲逛 + 夜景 + 招牌打卡）

**市内交通**：地铁为主，建议使用大阪地铁一日券 / ICOCA
**住宿**：大阪难波 / 日本桥

---

# Day 2 大阪休闲半日 + 前往[[奈良]] / [[京都]]

1. 上午：自由补逛（可选：梅田景观 / 天王寺 / 购物）
2. 中午：退房、寄存行李
3. 下午：出发前往 **[[奈良]]** 或 **[[京都]]**
    - [[大阪]]→[[奈良]]：近铁直达，约 45 分钟
    - [[大阪]]→[[京都]]：JR / 近铁，约 30-45 分钟
        **交通**：直接使用 ICOCA 刷卡乘车

---

# 大阪行程极简贴士

1. 动线优先： [[大阪城公园]] → [[黑门市场]] → [[心斋桥-道顿堀]] 一条线最顺
2. 交通：一张 ICOCA 全程通用
