## 📅 前期准备与交通

### ✈️ 抵达路径

1. **长沙 → 香格里拉**：建议直飞或经昆明中转，以节省体能。
    
2. **香格里拉 → 飞来寺**：乘大巴或包车（约 3-4 小时）。建议在飞来寺住宿一晚，次日看“日照金山”。
    
3. **飞来寺 → 尼农**：包车前往西当徒步起点。
    

### 🎒 装备清单(轻装)

- [x] **鞋靴**：高帮防水登山鞋。 
    
- [x] **服装**：冲锋衣裤（硬壳）、羽绒内胆、速干衣、厚登山袜。 
    
- [x] **工具**：登山杖（双杖）、头灯、充电宝（3C）、38L小鹰背包。 
    
- [ ] **药品**：布洛芬、抗高反药物、创可贴、葡萄糖口服液。

## 🗺️ 5天徒步行程

| **天数** | **行程内容**             | **难度** | **亮点**    |
| ------ | -------------------- | ------ | --------- |
| **D1** | 飞来寺 - 尼农 - 上雨崩       | ⭐⭐     | 进山初体验     |
| **D2** | 上雨崩 - 冰湖 - 上雨崩       | ⭐⭐⭐⭐   | 梅里之心、高山海子 |
| **D3** | 上雨崩 - 下雨崩 - 神瀑 - 下雨崩 | ⭐⭐⭐    | 宗教朝圣、经幡林  |
| **D4** | 下雨崩 - 神湖 (可选)        | ⭐⭐⭐⭐⭐  | 极高难度、无人之境 |
| **D5** | 下雨崩 - 尼农峡谷 - 香格里拉    | ⭐⭐     | 峡谷景观、出山   |

## 📍 核心路线详解

### 1. 冰湖路线 (Ice Lake)

> [!abstract] 关键信息
> 
> - **距离**：往返约 12km
>     
> - **爬升**：约 700m
>     
> - **贴士**：经过笑农大本营时可稍作休息，最后一段碎石坡较陡。
>     

### 2. 神瀑路线 (Sacred Waterfall)

> [!todo] 朝圣体验
> 
> - 这是雨崩最神圣的路线，藏民相信在瀑布下淋水可以洗净罪孽。
>     
> - 全程水泥路或平整土路较多，难度适中。
>     

### 3. 出山：尼农峡谷

> [!warning] 注意安全
> 
> - 尼农线虽多为下坡，但一边是峭壁一边是深谷。
>     
> - **严禁**在雨天走此路，会有落石风险。
>
